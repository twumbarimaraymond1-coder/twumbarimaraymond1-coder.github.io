# twumbarimaraymond1-coder.github.io

Personal website of **Raymond Twum-Barima** — MEng Mechanical Engineering (Loughborough),
covering thermofluids, computational engineering and software projects.

A static site (plain HTML + CSS, no build step) served via GitHub Pages.

## Structure

- `index.html` — home
- `about.html` — about
- `projects.html` — projects index
- `project-*.html` — individual case studies (boiling-heat-transfer ML, CFD, heat transfer, FEA, sustainability, UKAEA placement, GraceMeet)
- `experience.html` — experience & education
- `assets/` — `style.css`, `script.js`, images, CV and report PDFs

## Local preview

```bash
python -m http.server 8099
# then open http://localhost:8099
```
